# Contributing

1. Create a focused branch.
2. Use synthetic data only.
3. Run `bash scripts/security-check.sh`.
4. Confirm examples still follow the `AGENT.md`, `CONTEXT.md`, and
   `REFERENCES.md` contract.
5. Open a pull request describing the workflow behavior being changed.

Do not include real customer names, contact details, documents, screenshots,
emails, schedules, prices, credentials, or production exports.
