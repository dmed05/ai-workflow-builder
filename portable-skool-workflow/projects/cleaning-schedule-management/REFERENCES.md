# References

## Notes

This is a sanitized workflow skeleton. Keep customer names, active schedules, photos, documents, and private calendar exports out of GitHub unless explicitly approved.

## Common Status Files For Private Local Use

Private local versions may use files such as:

- `KEY_FINDINGS_SUMMARY.md`
- `SCHEDULING_ACTIONS.md`
- `FOLLOW_UP_ACTION_ITEMS.md`

Do not publish those files if they contain customer names or live schedule details.
