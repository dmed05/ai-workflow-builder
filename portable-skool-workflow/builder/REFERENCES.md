## Main Template

Use `../../AI Agent Problem Workflow Template.md` as the source template for creating new problem-specific workflow folders.

## Good Example

A good first version has:

- one clearly named problem folder
- one short `AGENT.md`
- one practical `CONTEXT.md`
- one minimal `REFERENCES.md`
- one dry test prompt

## Notes

Start simple. Use the system. Improve only after real use shows what is missing.
